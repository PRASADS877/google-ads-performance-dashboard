# google-ads-performance-dashboard
Interactive Power BI dashboard analyzing Google Ads performance across ad-level and campaign-level views

Interactive Power BI dashboard analyzing 5,000 Google Ads records across a 12-month window (May 2024–May 2025).

## Overview
- **Ad-Level View**: CTR, Conversion Rate, ROAS, Cost per Conversion, Top/Bottom 5 ad group comparisons
- **Campaign-Level View**: Cost share breakdown, monthly trend analysis, CTR vs target gauge

## Features
- Custom DAX measures (CTR, ROAS, CPM, Conversion Rate)
- Date table with time intelligence
- Drill-through from Campaign to Ad level
- Interactive slicers (Year, Month, Ad Group)
- Custom tooltips

## Key Insight
Flagged an unusually high ROAS (~230x) during analysis, traced back to how conversion values were scaled in the sample dataset — noted as a data quality observation rather than presented at face value.

## Tools Used
Power BI (Power Query, DAX, Data Modeling)
